範例通過 ECharts，來實現圖表數據資訊的展示。ECharts 是一個使用 JavaScript 實現的開源視覺化庫，可提供直觀，互動豐富，可高度個性化定製的數據視覺化圖表。

通過範例學習，可以學習在 Smart 中引入 ECharts 進行圖表展示。
